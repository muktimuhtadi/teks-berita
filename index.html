Tentang Situs Web

Situs web ini dikembangkan oleh Mukti Muhtadi - mahasiswa Program Studi Pendidikan Bahasa dan Sastra Indonesia, Fakultas Bahasa, Seni, dan Budaya, Universitas Negeri Yogyakarta - dengan tujuan digunakan sebagai media pembelajaran untuk kelas XI SMK Fase F Tingkat Lanjut pada mata pelajaran Bahasa Indonesia Kurikulum Merdeka. Semoga situs web media pembelajaran ini dapat memberi manfaat bagi banyak pihak. Kritik dan saran mengenai situs web ini, silakan sampaikan melalui email atau whatsapp tertera.
